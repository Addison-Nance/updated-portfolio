Train SchedulerMore

Deployed App Github Repo

Train Schedulerclose

This was my first project using the MomentJs library and the Firebase database. Using MomentJs to calculate times, and Firebase to store all the data, I created a very simple mock scheduler. The user can input details about the train such as it's name, first arrival time, and and frequency, which is then all stored in a Firebase database. The information is written to the DOM so that any trains added will always display and when a new train is added, it updates.
